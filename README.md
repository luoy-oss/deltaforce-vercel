DeltaForce API
===============

## 特性

* 基于PHP`8.0+`

> DeltaForce API的运行环境要求PHP8.0+

## 文档

[API文档](https://doc.apipost.net/docs/445c633fa47c000?locale=zh-cn)


## 安装

~~~
composer create-project coolxitech/deltaforce deltaforce
~~~

启动服务

~~~
cd deltaforce
php think run
~~~

然后就可以在浏览器中访问

~~~
http://localhost:8000
~~~

如果需要更新框架使用
~~~
composer update coolxitech/deltaforce
~~~
